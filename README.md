# 🍽️ Restaurant Website

## 📖 Description

Restaurant Website is a modern and responsive website built using HTML and CSS. It allows users to explore the restaurant's menu, view special dishes, learn about the restaurant, make reservations, and contact the restaurant.

## 🚀 Features

* Responsive Navigation Bar
* Home Page Banner
* About Us Section
* Food Menu
* Featured Dishes
* Chef Specials
* Online Reservation Form
* Customer Reviews
* Contact Information
* Gallery Section
* Footer with Social Media Links
* Attractive Hover Effects

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
